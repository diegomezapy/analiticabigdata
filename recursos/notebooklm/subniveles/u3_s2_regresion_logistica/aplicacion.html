3.2 Aplicación: Modelo de Scoring de Crédito

Contexto

El scoring de crédito es una de las aplicaciones más clásicas de la regresión logística en la industria financiera. Instituciones como Bancos y Cooperativas en Paraguay usan estos modelos para decidir si otorgar préstamos.

Implementación completa

library(glmnet); library(pROC); library(caret)
set.seed(77)
n <- 2000
creditos <- data.frame(
  edad = round(rnorm(n, 35, 10)),
  ingreso_log = log(rexp(n, 0.0000001) + 1000000),
  deuda_ratio = pmin(1, pmax(0, rnorm(n, 0.35, 0.2))),
  antiguedad_trabajo = round(rexp(n, 0.25)),
  num_creditos_previos = rpois(n, 2),
  tiene_garantia = sample(c(0,1), n, replace=TRUE, prob=c(0.4,0.6))
)
lp <- -4 + 0.03*creditos$edad + 0.5*creditos$ingreso_log -
      2.5*creditos$deuda_ratio + 0.1*creditos$antiguedad_trabajo +
      0.2*creditos$num_creditos_previos + 0.8*creditos$tiene_garantia
creditos$buen_pagador <- factor(rbinom(n,1,1/(1+exp(-lp))), labels=c("No","Si"))

# Pipeline
idx <- createDataPartition(creditos$buen_pagador, p=0.8, list=FALSE)
train <- creditos[idx,]; test <- creditos[-idx,]

# Modelo con Lasso (selección automática de variables)
X_train <- model.matrix(buen_pagador~., train)[,-1]
y_train <- as.numeric(train$buen_pagador=="Si")
X_test  <- model.matrix(buen_pagador~., test)[,-1]

lasso_cv <- cv.glmnet(X_train, y_train, family="binomial", alpha=1, nfolds=10)
cat("Lambda óptimo:", lasso_cv$lambda.min, "
")
coef(lasso_cv, s="lambda.min")  # Variables seleccionadas

# Score crediticio (probabilidad de buen pagador)
scores <- predict(lasso_cv, X_test, s="lambda.min", type="response")
roc_res <- roc(test$buen_pagador, as.vector(scores))
cat("AUC:", round(auc(roc_res),3), "
")
plot(roc_res, main="Curva ROC — Scoring de Crédito")
Regulación: En muchos países, los modelos de crédito deben ser "explicables" (no cajas negras). La regresión logística cumple con este requisito ya que sus coeficientes/odds ratios son interpretables.
