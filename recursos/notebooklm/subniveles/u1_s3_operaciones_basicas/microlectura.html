1. El ciclo de exploración

Todo análisis de datos sigue un ciclo de 4 pasos:

  1. Preguntar: ¿Qué quiero saber? (ej: ¿cuál región tiene mayor prevalencia de diabetes?)
  2. Filtrar/Agregar: Seleccionar y resumir los datos necesarios.
  3. Visualizar: Representar el patrón visualmente.
  4. Interpretar: ¿Qué dice el dato? ¿Qué decisión informaría?

2. Operaciones en data.table (R)

La sintaxis de data.table es dt[i, j, by]:

  • i: filas a seleccionar (filtro)
  • j: columnas o cálculos a realizar
  • by: variable de agrupación

Esta sintaxis concisa es extremadamente eficiente para datasets grandes.

3. Visualización con ggplot2

ggplot2 usa una "gramática de gráficos": cada gráfico se construye con capas:

  1. ggplot(data, aes(x=..., y=..., color=...)): datos y mapeo estético
  2. geom_*(): tipo de gráfico (geom_histogram, geom_boxplot, geom_point)
  3. labs(): títulos y etiquetas
  4. theme_*(): estilo visual
