Subnivel: 1.3 Operaciones básicas, resúmenes y gráficos

Procedimiento: 1.3 Operaciones básicas

  1. Cargar datos con fread() o pl.read_csv(). Ejecutar diagnóstico inicial.
  2. Formular preguntas: definir al menos 3 preguntas analíticas sobre los datos (ej: ¿cuál región tiene mayor tiempo de espera promedio?).
  3. Filtrar y agregar: para cada pregunta, escribir el código de filtrado/agrupación correspondiente.
  4. Visualizar: crear al menos un gráfico por pregunta. Elegir el tipo adecuado: histograma (distribución), boxplot (comparación grupos), barras (frecuencias), scatter (relaciones).
  5. Interpretar: escribir 1-2 oraciones de conclusión por cada gráfico. Incluir valores numéricos concretos.
  6. Guardar salidas: usar ggsave() o plt.savefig() para exportar gráficos como PNG a outputs/graficos/.
  7. Documentar: comentar cada bloque de código explicando qué se calcula y por qué.
