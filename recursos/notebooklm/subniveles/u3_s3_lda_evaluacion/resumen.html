3.3 LDA y Evaluación de Modelos — Resumen

LDA (Linear Discriminant Analysis) es a la vez un método de clasificación y reducción de dimensionalidad. La evaluación de modelos provee un framework riguroso para comparar algoritmos.

LDA

LDA encuentra las combinaciones lineales de features que maximizan la separación entre clases. Supuestos: distribución normal multivariada y matrices de covarianza iguales entre clases.

  • Between-class scatter (SB): varianza entre las medias de las clases.
  • Within-class scatter (SW): varianza dentro de cada clase.
  • Objetivo: maximizar SB/SW — maximizar separación relativa a dispersión interna.
  • QDA: versión cuadrática que permite matrices de covarianza distintas por clase.

Validación cruzada k-fold

Divide el dataset en k partes iguales. Itera k veces: una parte como test, resto como entrenamiento. Retorna k estimaciones de rendimiento → media y desvío.
