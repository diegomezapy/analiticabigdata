4.1 Importancia de Variables y Feature Selection — Resumen

La importancia de variables cuantifica cuánto contribuye cada feature a las predicciones del modelo. La selección de features (feature selection) identifica el subconjunto óptimo para mejorar rendimiento, reducir overfitting y aumentar interpretabilidad.

Métodos de importancia de variables

MétodoDescripciónModelos aplicables

Feature Selection

  • Filter methods: seleccionan features por estadísticos (correlación, chi-cuadrado, ANOVA) independiente del modelo.
  • Wrapper methods: RFE (Recursive Feature Elimination) — entrena el modelo, elimina la feature menos importante, repite.
  • Embedded methods: regularización L1/Lasso — elimina features durante el entrenamiento.

Regla general

Más features no siempre = mejor modelo. La maldición de la dimensionalidad: en espacios de alta dimensión, los datos se vuelven dispersos y los algoritmos basados en distancia (KNN) se degradan. Generalmente, un modelo con 10 features bien seleccionadas supera a uno con 50 features sin selección.
