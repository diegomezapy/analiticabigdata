2.3 Correlación y Análisis de Agrupación — Resumen

El análisis de correlación mide la fuerza y dirección de la relación lineal entre dos variables. El análisis de agrupación (groupby) resume datasets por categorías para obtener métricas por segmento.

Correlación de Pearson

r = Cov(X,Y) / (σX × σY), donde r ∈ [−1, 1].

  • r = 1: correlación positiva perfecta
  • r = 0: sin correlación lineal
  • r = −1: correlación negativa perfecta
  • |r| > 0.7: fuerte; 0.4–0.7: moderada; < 0.4: débil
Correlación ≠ Causalidad: Variables pueden correlacionar por coincidencia o por una variable confusora común.

Correlación de Spearman

Alternativa no paramétrica: mide correlación entre los rangos de las variables. Más robusta a outliers y para variables ordinales.

Agrupación con dplyr

df %>%
  group_by(categoria) %>%
  summarise(
    n = n(),
    media = mean(valor),
    sd = sd(valor),
    mediana = median(valor)
  )

Tabla cruzada (contingencia)

Para variables categóricas: table(df$var1, df$var2) o crosstab en Python. Chi-cuadrado prueba independencia.
