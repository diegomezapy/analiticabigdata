2.3 Práctica Guiada: Análisis de Correlación en Dataset Económico

Duración: 50 minutos | Dataset: World Bank indicators

Ficha 1: Simular datos de indicadores económicos

set.seed(21)
n <- 50
economicos <- data.frame(
  pais = paste0("Pais_", 1:n),
  region = sample(c("América","Europa","Asia","Africa"), n, replace=TRUE),
  pib_per_capita = exp(rnorm(n, 8.5, 1.5)),  # log-normal
  esperanza_vida = rnorm(n, 68, 10),
  tasa_alfabetizacion = pmin(100, pmax(40, rnorm(n, 80, 15))),
  co2_per_capita = exp(rnorm(n, 1.5, 1)),
  gasto_educacion_pct = pmax(0, rnorm(n, 5, 2))
)
head(economicos, 3)

¿Qué tipo tiene cada variable? ¿Qué rango esperarías para el PIB per cápita?

Ficha 2: Matriz de correlación

library(corrplot)
vars_num <- economicos[, c("pib_per_capita","esperanza_vida",
                            "tasa_alfabetizacion","co2_per_capita","gasto_educacion_pct")]
cor_mat <- cor(vars_num)
corrplot(cor_mat, method = "color", addCoef.col = "black",
         tl.cex = 0.7, number.cex = 0.7)

¿Qué par de variables tiene la correlación más alta? ¿Qué par tiene correlación negativa?

Ficha 3: Agrupación por región

library(dplyr)
economicos %>%
  group_by(region) %>%
  summarise(
    n_paises = n(),
    pib_mediana = median(pib_per_capita),
    vida_media = mean(esperanza_vida),
    alfabetizacion = mean(tasa_alfabetizacion)
  ) %>%
  arrange(desc(pib_mediana))

¿Qué región tiene mayor PIB mediano? ¿Coincide con mayor esperanza de vida?

Ficha 4: Gráfico de burbuja

library(ggplot2)
ggplot(economicos, aes(x = pib_per_capita, y = esperanza_vida,
                        size = tasa_alfabetizacion, color = region)) +
  geom_point(alpha = 0.7) +
  scale_x_log10(labels = scales::dollar) +
  scale_size_continuous(range = c(3, 12), name = "Alfabetización %") +
  labs(title = "PIB vs Esperanza de vida",
       x = "PIB per cápita (escala log)", y = "Esperanza de vida (años)") +
  theme_minimal()

¿Qué patrón observas? ¿La relación es lineal o curvilínea en escala original?
