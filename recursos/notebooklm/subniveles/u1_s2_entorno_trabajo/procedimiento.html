Subnivel: 1.2 Entorno de trabajo: R/RStudio, archivos y lectura de datos

Procedimiento: 1.2 Configurar entorno y leer datos

  1. Instalar R ≥ 4.3 desde cran.r-project.org. Verificar con R.version.
  2. Instalar RStudio ≥ 2024 desde posit.co. Abrir RStudio y verificar que R está conectado (esquina inferior izquierda de la consola).
  3. Crear proyecto RStudio: File → New Project → New Directory. Nombrar con patrón ABD_UNIDAD1.
  4. Crear estructura de carpetas ejecutando en consola: 
    dir.create("data/raw", recursive=TRUE); dir.create("data/processed"); dir.create("scripts"); dir.create("outputs")
  5. Instalar paquetes esenciales: ejecutar el bloque de instalación de la guía de funciones.
  6. Cargar el dataset con la función correspondiente al formato (.csv → fread, .xlsx → read_excel, .parquet → read_parquet).
  7. Ejecutar diagnóstico: dim(), str(), summary(), colSums(is.na(df)).
  8. Documentar hallazgos en un comentario al inicio del script: filas, columnas, tipos problemáticos, columnas con NAs.
